Web Crawler

This app with crawl the web, starting from a web page of your choice. The crawler will follow every "valid" link on every web page it visits, until reaching the specified traversal depth or the maximum number of nodes. If you enter a keyword, the crawler will keep track of which pages contain that keyword. After the crawler is finished, it will display the results as a graph. The starting page ("node") and the web pages that contain the specified keyword will be highlighted differently from the others.

Note that at this time the web crawler is in an early stage of development. While functional, it is not yet optimized to crawl large numbers of web pages quickly. It is therefore recommended that you start with a small number of nodes, perhaps around 10, to avoid long delays and potential timeout issues. Furthermore, Google Chrome version 56 is the only browser that has been tested against. It is known that there are small yet annoying compatibility issues with Microsoft Edge and Mozilla Firefox.
